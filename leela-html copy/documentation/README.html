B. HTML Structure

“Leela” html template built on Zurb Foundation 5 framework for easy editing. Please check additional documentation on Zurb Foundation website.

This theme is a responsive layout. Default template has 12 sections:

  1. Navigation
  2. Header
  3. Overview
  4. Our Team
  5. Portfolio
  6. Project (hidden section by default, but opens when user click on one of potfolio item)
  7. Testimonials
  8. One More (this section is used for example purpose)
  9. Clients
  10. Contact
  11. Map
  12. Footer

C. CSS Files and Structure

I'm using four CSS files in this theme.

  • font-awesome.css font awesome font and icons
  • app.css foundation 5
  • component.css effects and animation
  • theme.css theme stylesheet

The theme.css contains all of the specific stylings for the page. The file is separated into sections using:


/*** Sections ***/

some code

/*** Section Purple ***/

some code

/*** Section Dark Blue ***/

some code

/*** Section Light Rose ***/

some code

etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

D. JavaScript

This template uses the following JavaScript files, located in the js folder.

  • Foundation 5 javascript library
  • jQuery and some plugins:
    • jQuery Waypoints (this plugin makes it easy to execute a function whenever you scroll to an element)
    • animOnScroll (jQuery plugin that animates elements based on scrollbar position)
    • Masonry (for portfolio grid)
    • Headroom.js (for “sticky” navigation)
  • Additional javascript libraries:
    • imagesloaded.js
    • classie.js
    • toucheffects.js
    • Google Maps API
  • app.js (call and activate contains plugins, small functions)
How to edit Google Maps address location and change Google Maps theme

1. Open js/app.js file

2. Scroll down and on line 165 you will see Google Maps theme options. On that place you can modify colors and visibilty for some layers.


// Google Maps Theme

var googlemaptheme = [
  {
    "featureType": "water",
    "stylers": [
      {
        "visibility": "on"
      },
      {
        "color": "#acbcc9"
      }
    ]
  },
  {
    "featureType": "landscape",
    "stylers": [
      {
        "color": "#f2e5d4"
      }
    ]
  }
  ...
];

3. On line 266 and 273 you will be able to set up coordinates and address location.


// Google Maps Initialize

function initialize() {
  var mapOptions = {
    zoom: 9,
    center: new google.maps.LatLng(37.42265,-122.163417),
    scrollwheel: false,
    mapTypeControl: false,
    styles: googlemaptheme
  }

  var map = new google.maps.Map(document.getElementById('google-map'), mapOptions);
  var myLatLng = new google.maps.LatLng(37.42265,-122.163417);
  var image = 'images/google-maps/pin.png';

E. PHP Contact Form

How to set up PHP Contact Form.

1. Open file php/form.php

2. On line 22 paste your email address


$email_to = "place@your.email"; // place your email here

That's it.
